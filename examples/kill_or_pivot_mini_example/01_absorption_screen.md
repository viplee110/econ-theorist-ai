# 01 Absorption Screen

Status: Toy example only

## Candidate Claim

Early public recognition raises team output.

## Absorption Risks

Risk 1: Social-image absorption

- If recognition only increases reputational utility, the result is a standard
  social-image comparative static.
- Required escape: recognition timing must change task allocation, not merely
  effort level.

Risk 2: Career-concerns absorption

- If public signals only update beliefs about worker ability, the result is a
  career-concerns model.
- Required escape: the theorem must depend on visibility of task margins, not
  only posterior beliefs.

Risk 3: Multitask moral-hazard absorption

- If visible and hidden tasks only create a standard incentive tradeoff, the
  result may be a relabeling.
- Required escape: recognition timing must alter which tradeoff is operative.

## Preliminary Decision

Do not write a manuscript. Run a verified literature audit and D4.5 model-base
reset unless the primitive-theorem pair becomes sharper.
