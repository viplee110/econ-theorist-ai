# Kill Or Pivot Mini Example

This example shows why the system is not just a paper generator. A promising
idea can be killed, parked, or pivoted when evidence shows weak novelty, weak
theorem bite, or excessive assumption debt.

Toy idea:

```text
Early public recognition raises team output.
```

Expected behavior:

- Run literature evidence checks before novelty claims.
- Ask whether the result is merely social-image theory, career concerns, or
  standard multitask moral hazard under new language.
- Stop manuscript development if absorption risk remains unresolved.
