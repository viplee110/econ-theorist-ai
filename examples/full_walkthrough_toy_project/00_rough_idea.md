# 00 Rough Idea

Status: Toy example only

User prompt:

```text
Use the system: I want to explore a theory idea about public recognition in
team production. I do not yet know the right model.
```

Expected route:

```text
D0/D1 open discovery -> literature evidence ledger -> field profile ->
target journal profile -> D4 primitive hunting -> D4.5 model base
```

Why this is discovery mode:

- The user named a field and an economic tension, not a confirmed model.
- Any proposed agents, timing, payoffs, tasks, or recognition objects remain
  provisional.
- The system should not jump to fixed points, contraction mappings, or a full
  equilibrium model before the smallest economic example is clear.

Initial candidate question:

```text
Can public recognition improve team output when it motivates visible effort, but
reduce output when it causes workers to shift away from less visible but more
productive tasks?
```

Initial danger:

This may be absorbed by existing moral hazard, career concerns, or social image
models unless the primitive-theorem pair is sharper.
